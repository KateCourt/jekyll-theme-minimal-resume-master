OrQA: Organ Quality Assessment is a multidisciplinary project involving academics, clinicians and software engineers. The aim of this project is to leverage moderm machine learning techniques to generate robust automated predictions of transplant viability from images of candidate organs. If you are involved in organ transplant and would like to learn more about how you can get involved please get in touch.

Generate training data

We are developing an application to facilitate transplant experts to view and assess images of organs for transplantability. We are currently looking for transplant professionals to get involved, so if you have the expertise to help we would love to hear from you!

Develop machine learning models

Images of organs that have been assessed by experts can then be used to train cutting-edge machine learning models to provide an automated assessment.

Organ Quality Scoring

The main deliverable of this project will be a tool that allows surgeons to upload a photograph of an organ and receive an automated assessment of how viable the organ is for transplant. This is intended to support decision making around transplantation, in the hopes that we can increase the number of successful transplants in the UK and beyond.
